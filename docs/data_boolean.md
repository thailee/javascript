#Boolean

Boolean has only 2 values: true and false

```javascript
console.log("Abc" < "Abcd") // -> true
console.log("abc" < "Abcd") // -> false
console.log("123" == "123") // -> true
console.log(NaN == NaN) // -> false
```

What is the different?

```javascript
console.log("5" == 5);
console.log("5" === 5);
```
